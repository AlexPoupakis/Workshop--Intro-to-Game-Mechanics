# Workshop: Intro to Game Mechanics
A repository for my 2-hour "Intro to Game Mechanics" Workshop in the Startup Greece Week 2020 event. The goal was to get familiar with basic (2D) game concepts and implement (part of) a 2D browser game.

## Overview
* Game loops
* Affine Matrix Transformations
* Animation sequencing
* Collision Detection
* Parallax scrolling
